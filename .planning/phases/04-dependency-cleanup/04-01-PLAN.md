# Phase 4: Dependency Cleanup

## Goal
Remove all Sarvam AI and OpenVoice code and packages; `uv sync` resolves a clean, minimal dependency set.

## Success Criteria
1. `uv sync` completes with no `sarvamai`, `wavmark`, `langid`, `whisper-timestamped`, `inflect`, `unidecode`, `eng-to-ipa`, `pypinyin`, `jieba`, or `cn2an` packages installed
2. `src/indic_voice/openvoice/` directory no longer exists
3. `tts_sarvam.py`, `models/tone_transfer.py`, and `models/checkpoint_manager.py` no longer exist
4. `uv run indic-voice clone --text "नमस्ते" --ref-voice ref.wav` still succeeds after all removals
5. `uv lock` is regenerated and committed; lockfile contains no removed packages

## Requirements (DEPS-01 through DEPS-05)
- [DEPS-01] No removed packages in pyproject.toml
- [DEPS-02] openvoice/ directory removed
- [DEPS-03] Legacy files removed (tone_transfer.py, checkpoint_manager.py, tts_sarvam.py)
- [DEPS-04] CLI still works after removals
- [DEPS-05] Lockfile regenerated

## Execution

### Step 1: Remove packages from pyproject.toml
Delete:
- `sarvamai==0.1.0`
- `wavmark>=0.0.3`
- `langid>=1.1.6`
- `whisper-timestamped>=1.15.4`
- `inflect>=5.6.0`
- `unidecode>=1.3.6`
- `eng-to-ipa>=0.0.2`
- `pypinyin>=0.50.0`
- `jieba>=0.42.1`
- `cn2an>=0.5.22`

### Step 2: Delete files/directories
- `src/indic_voice/openvoice/` (entire directory)
- `src/indic_voice/models/tone_transfer.py`
- `src/indic_voice/models/checkpoint_manager.py`
- `src/indic_voice/pipeline/tts_sarvam.py`

### Step 3: Verify no remaining imports of removed packages
```bash
grep -r "sarvam\|openvoice\|tone_transfer\|checkpoint_manager\|wavmark\|langid\|inflect\|unidecode\|eng-to-ipa\|pypinyin\|jieba\|cn2an\|whisper-timestamped" src/
```
Should return no matches.

### Step 4: Regenerate lockfile
```bash
uv lock
```

### Step 5: Verify CLI still works
```bash
uv run indic-voice --help
uv run indic-voice clone --text "नमस्ते" --ref-voice ref.wav
```

### Step 6: Verify tests pass
```bash
uv run pytest tests/ -v
```
